# Sauce-Demo-Automation-Framework-using-Playwright
A POM-based end-to-end test automation framework for Sauce Demo built with Playwright, supporting Firefox, Chromium, and WebKit browsers. This project demonstrates a complete e-commerce checkout flow using modern automation best practices.

**Features**

Page Object Model (POM) design for modular and maintainable tests

Login automation for valid users

Product selection and add-to-cart functionality

Cart verification to ensure selected products are added correctly

Checkout process including user details and final order completion

Cross-browser support: Firefox, Chromium, WebKit

Built-in waits and synchronization to handle dynamic page elements

Screenshots, videos, and traces for debugging failed tests
